# CurrencyConverter

Простой конвертер валют, написанный на Swift 5.
Обменные курсы подгружаются из: https://site.financialmodelingprep.com/developer/docs/#Forex-(FX)

Стек технологий:
* Swift
* Uikit
* CoreData

Архитектура: 
      MVP
      
Библиотеки:
* SnapKit
* Charts
